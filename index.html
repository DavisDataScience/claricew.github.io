We are the Data Science Association at UC Davis.

Mission Statement

What we do

  • Unite students of all disciplines through promoting a basic proficiency in data analysis
  • Partnership with the Data Science Initiative at Davis, an organization geared towards grad students, faculty, and companies
  • Collaborate with other data science organizations at other UCs to host joint competitions and cultivate cross-campus interest in data science
  • host Ted Talks, mentoring events, and mixers to provide opportunities for students to meet skilled scientists in the industry or conducting research
  • Foster an environment for our club members’ improvement of data skills with workshops, seminars, and end-of-the-quarter competitions
  • Our workshops will include topics ranging from programming to basic statistical concepts, for beginners and advanced data miners alike

Our club members

  • Enjoy problem-solving, asking questions (Creativity), and are willing to learn the tools of the trade
  • Are self-motivated while being a team player, and love projects
  • Come from a diverse range of majors!

Personal Projects


ArduPi-ECG

Project using Python , C for the Arduino Pulse Sensor to efficiently analyze and predict heart rate/pulse data, using Chaotic, Signal Extrapolation, Deep Neural Network, Linear Regression models. Github link


Chaotic Neuron

Using chaotic time intervals from the logistic map equation as input to Hodgkin-Huxley, Fitzhugh-Nagumo, and LIF neuron models in order to simulate real-world random stimuli. Github link


Tequila-Mockingbird

Program that can 'listen' to any short melody and identify notes, create its own original song from it using Python Tensorflow. Run on Raspberry Pi 3 with Google Cloud Voice. Responds only to the word 'Tequila' to initiate the program ;) Github link
